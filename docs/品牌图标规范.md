# Cursor Skin Manager 品牌图标规范

## 唯一母版

项目 Logo 以 `src-tauri/icons/icon-master.png` 为唯一视觉母版。该图片包含：

- 蓝色圆角方形外侧为完全透明背景
- 蓝色圆角方形
- 左上角青绿色圆点
- 中央白色鼠标指针

不得重新描摹、调整比例、移动元素、恢复白色外框或使用旧版 SVG 替代。

## 生成资源

运行 `scripts/generate-icon.ps1` 后生成：

- `src-tauri/icons/icon.ico`：Windows EXE、窗口、任务栏、托盘和安装包图标
- `src-tauri/icons/icon-16.png` 至 `icon-256.png`：各档高清 PNG
- `src-tauri/icons/icon-preview.png`：512px 预览
- `public/favicon.png`：开发页面 favicon

Windows ICO 固定包含 16、20、24、32、40、48、64、128、256px 九档资源。后续版本必须运行生成脚本统一更新，禁止只替换单个文件。

生成时会裁掉母版中过量的透明画布，并按 Windows 光学尺寸重新留边：16/20px 接近满幅显示，32px 以上保留约 4%–5% 安全边距。不得直接缩放整张母版，否则图标会比其他应用明显偏小。
